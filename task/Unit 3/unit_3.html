Tarea 7

Al pulsar el botón de "añadir al carito" en "Task 6" tienes que mostrar el carito en una tabla con estos datos en .out-7.

En esta tarea, se te pide crear un carrito tipo tabla de compra. El carrito debe permitir al usuario:

  • Ver el contenido del carrito: El usuario debe poder ver una lista de los artículos que ha agregado al carrito, incluyendo el título, la cantidad y el precio de cada artículo.
  • Modificar la cantidad de artículos: El usuario debe poder modificar la cantidad de cada artículo en el carrito.
  • Eliminar artículos del carrito: El usuario debe poder eliminar artículos del carrito.
  • Ver el precio total del carito.
  • Botones de acción: + & - para cantidad.

Importante: Los datos no se tienen que perder con cerrar el navegador.

En out-7 mostrar el carito en una tabla

Tarea 8

Crear un botón "Tramitar pedido" en "Task 7": Debes mostrar el botón debajo de la tabla y con un anclaje llevar a out-8 al formulario.

En esta tarea, se te pide crear un formulario para recopilar los datos necesarios para crear un pedido de venta utilizando la API de A3 Documentos/PV.

  • Cree el formulario con los datos necesarios. Investigue en la documentación de la API para determinar qué campos son necesarios.
  • Implementar control de errores. Maneja los posibles errores que puedan ocurrir al enviar el formulario o al procesar el pedido.
  • Mostrar un mensaje de éxito. Si el pedido se ha procesado correctamente, muestra un mensaje en out-8 indicando que el pedido está tramitado.
